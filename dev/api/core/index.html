Relevant graph projections

SBMLGraphs.projected_graph — Method
projected_graph(A::AbstractMatrix{T}, Vp::AbstractVector{Int}) where T

Returns the projection of A onto one of its node sets.

Returns the adjacency matrix Ap that is the projection of the bipartite graph adjacency matrix A onto the specified nodes. They are connected in Ap if they have a common neighbor in A.

Arguments

  • A::AbstractMatrix{T}: Adjacency matrix of the bipartite graph.
  • Vp::AbstractVector{Int}: List of nodes to project onto.

Returns

  • AbstractMatrix{T}: A graph adjacency matrix that is the projection onto the given nodes.

Extended help

This algorithm doesn't check if the inputted adjacency matrix is for a bipartite graph, therefore the algorithm will fail whenever it detects it's not bipartite, resulting in useless computation time.
